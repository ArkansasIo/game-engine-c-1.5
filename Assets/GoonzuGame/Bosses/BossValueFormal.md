# Boss Value Formal

| Property   | Type    | Description                  |
|------------|---------|------------------------------|
| Name       | string  | Boss name                    |
| Level      | int     | Boss level                   |
| Health     | int     | Boss health                  |
| Attack     | int     | Boss attack                  |
| Defense    | int     | Boss defense                 |
| Abilities  | list    | Boss abilities               |
| Theme      | string  | Boss theme                   |

Example:

- Name: Dragon King
- Level: 100
- Health: 50000
- Attack: 1200
- Defense: 800
- Abilities: [Fire Breath, Tail Swipe]
- Theme: Fire
