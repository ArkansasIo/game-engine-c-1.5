# Armor Value Formal

| Property   | Type    | Description                  |
|------------|---------|------------------------------|
| Name       | string  | Armor name                   |
| Defense    | int     | Defense value                |
| Durability | int     | Durability value             |
| PvPType    | string  | PvP type (Plate, Leather)    |
| PvEType    | string  | PvE type (Plate, Leather)    |
| Rarity     | string  | Armor rarity                 |
| Theme      | string  | Armor theme                  |

Example:

- Name: Plate of Light
- Defense: 60
- Durability: 100
- PvPType: Plate
- PvEType: Plate
- Rarity: Rare
- Theme: Light
