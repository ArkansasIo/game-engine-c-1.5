# Mob Value Formal

| Property   | Type    | Description                  |
|------------|---------|------------------------------|
| Name       | string  | Mob name                     |
| Level      | int     | Mob level                    |
| Health     | int     | Mob health                   |
| Attack     | int     | Mob attack                   |
| Defense    | int     | Mob defense                  |
| Abilities  | list    | Mob abilities                |
| Theme      | string  | Mob theme                    |

Example:

- Name: Goblin Scout
- Level: 10
- Health: 500
- Attack: 40
- Defense: 20
- Abilities: [Quick Strike]
- Theme: Forest
