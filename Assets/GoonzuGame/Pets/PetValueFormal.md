# Pet Value Formal

| Property   | Type    | Description                  |
|------------|---------|------------------------------|
| Name       | string  | Pet name                     |
| Species    | string  | Pet species                  |
| Level      | int     | Pet level                    |
| Abilities  | list    | Pet abilities                |

Example:

- Name: Fluffy
- Species: Wolf
- Level: 5
- Abilities: [Howl, Bite]
