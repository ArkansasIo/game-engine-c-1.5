# Item Value Formal

| Property   | Type    | Description                  |
|------------|---------|------------------------------|
| Name       | string  | Item name                    |
| Rarity     | string  | Item rarity (Common, Rare)   |
| Theme      | string  | Item theme (Fire, Ice, etc.) |
| Description| string  | Item description             |
| Value      | int     | Item value/cost              |
| Stats      | object  | Item stats (Attack, Defense) |

Example:

- Name: Sword of Flames
- Rarity: Epic
- Theme: Fire
- Description: A sword imbued with fire magic.
- Value: 1000
- Stats: { Attack: 50, Defense: 10 }
