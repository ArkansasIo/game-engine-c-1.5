# Profession Value Formal

| Property   | Type    | Description                  |
|------------|---------|------------------------------|
| Name       | string  | Profession name              |
| Description| string  | Profession description        |
| Skills     | list    | Profession skills             |

Example:

- Name: Blacksmith
- Description: Forges weapons and armor
- Skills: [Forge, Repair, Upgrade]
