# Weapon Value Formal

| Property   | Type    | Description                  |
|------------|---------|------------------------------|
| Name       | string  | Weapon name                  |
| Attack     | int     | Attack value                 |
| Speed      | int     | Speed value                  |
| PvPType    | string  | PvP type (Sword, Bow, etc.)  |
| PvEType    | string  | PvE type (Sword, Bow, etc.)  |
| Rarity     | string  | Weapon rarity                |
| Theme      | string  | Weapon theme                 |

Example:

- Name: Bow of Shadows
- Attack: 40
- Speed: 20
- PvPType: Bow
- PvEType: Bow
- Rarity: Legendary
- Theme: Shadow
