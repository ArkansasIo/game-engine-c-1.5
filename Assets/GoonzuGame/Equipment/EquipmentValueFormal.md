# Equipment Value Formal

| Property   | Type    | Description                  |
|------------|---------|------------------------------|
| Name       | string  | Equipment name               |
| Slot       | string  | Equipment slot (Head, Chest) |
| Power      | int     | Power value                  |
| Durability | int     | Durability value             |
| Rarity     | string  | Equipment rarity             |
| Theme      | string  | Equipment theme              |

Example:

- Name: Helm of Thunder
- Slot: Head
- Power: 30
- Durability: 80
- Rarity: Epic
- Theme: Thunder
