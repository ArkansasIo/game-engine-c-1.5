# Enemy Value Formal

| Property   | Type    | Description                  |
|------------|---------|------------------------------|
| Name       | string  | Enemy name                   |
| Level      | int     | Enemy level                  |
| Health     | int     | Enemy health                 |
| Attack     | int     | Enemy attack                 |
| Defense    | int     | Enemy defense                |
| Abilities  | list    | Enemy abilities              |
| Theme      | string  | Enemy theme                  |

Example:

- Name: Undead Warrior
- Level: 25
- Health: 2000
- Attack: 120
- Defense: 60
- Abilities: [Curse, Shield Bash]
- Theme: Undead
