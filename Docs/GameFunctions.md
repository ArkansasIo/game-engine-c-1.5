# Game Functions

This document lists and describes key functions provided by the game engine.

## Core Functions
- `Initialize()`: Sets up engine
- `Run()`: Starts main loop
- `Shutdown()`: Cleans up resources
- `SaveGame(slot)`: Saves game state
- `LoadGame(slot)`: Loads game state
- `SetVolume(volume)`: Sets audio volume
- `SetKeybind(action, key)`: Customizes keybinds
- `SetControllerMapping(action, button)`: Customizes controller mapping

## Customization
Functions can be extended for additional features.
