# AI Asset Platforms Documentation

This folder contains documentation for various AI tools and platforms used for generating game assets for the GoonZu-style RPG/MMORPG engine.

## Files

- [All-in-One AI Asset Platforms](AllInOne.md) - Tools that generate multiple asset types
- [AI 3D Model Generators](AI3DModelGenerators.md) - 3D asset creation tools
- [2D Assets, Sprites & Textures](2DAssetsSpritesTextures.md) - 2D and pixel art tools
- [Free Asset Libraries](FreeAssetLibraries.md) - Non-AI asset sources
- [Tips and Workflow](TipsAndWorkflow.md) - Best practices and recommended workflows

## Integration with Project

Assets generated from these tools should be placed in the appropriate folders under `Assets/` (e.g., `Assets/GoonzuGame/` for game-specific assets).

For prompts and examples, see the main [README.md](../README.md) in the Docs folder.
