# Controls

This document describes the input controls for the game engine, including keyboard, mouse, and controller support (PS5/Xbox).

## Keyboard Controls
- Move Up: W
- Move Down: S
- Move Left: A
- Move Right: D
- Interact: E
- Open Menu: M

## Mouse Controls
- Left Click: Move
- Right Click: Interact

## Controller Controls
### Common Actions
- Move Up: D-pad Up
- Move Down: D-pad Down
- Move Left: D-pad Left
- Move Right: D-pad Right
- Interact: A/X (Xbox/PS5)
- Open Menu: Start/Options
- Jump: B/Circle
- Attack: X/Square
- Special: Y/Triangle
- Pause: Menu/Options

### PS5 Mapping
- Cross (X): Interact
- Circle: Jump
- Square: Attack
- Triangle: Special
- Options: Menu

### Xbox Mapping
- A: Interact
- B: Jump
- X: Attack
- Y: Special
- Menu: Pause

## Customization
Keybinds and controller mappings can be customized in-game.
